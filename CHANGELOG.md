## [1.0.0] - 2026-07-21

### 🚀 AÑADIDO
- Sistema de autenticación (registro/login de usuarios)
- Panel de administración para gestión de accesos
- Mapa interactivo con Leaflet
- Despliegue de activos aliados y enemigos
- Sistema de reportes en tiempo real
- Consola de operaciones
- Brújula táctica con orientación
- Exportación de mapa a PDF
- Base de datos en tiempo real con Firebase

### 🛠️ CORREGIDO
- Validación de cédula (7-8 dígitos)
- Validación de contraseña (8 caracteres, letras+números+símbolos)
- Restricción de despliegue solo en territorio venezolano

- README.md con instrucciones de instalación

- CHANGELOG.md con historial de cambios
